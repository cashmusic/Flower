Flower 1.0 (In Progress)

100% driven by lazy-loading JavaScript and a handful of CSS, with an emphasis on progressive enhancement.

The Flower Core loads, scans CSS classes and/or HREF information, loads required components, and remains available as a means of centralizing events. Both Moviebox and Imagebox extend a generic overlay class to display lightboxed images (loaded via class) and videos (loaded via HREF.)

The Soundplayer is an extensible playlist manager for SoundManager2 with intelligent switching of multiple playlists through one object instance. Audio can be loaded using a classed div, a classed anchor, or directly. The default UI is easily skinnable via CSS and can be fully overwritten via MooTools Class structures. A rich event structure can be used to do AJAX API authentication/requests for a tracked or secure player. Oh and it supports HTML5, has an iPhone-specific interface, and can pull in SoundCloud sets as playlists.

There are various other UI enhancements, some useful utility scripts, etc. Before 1.0 release we're adding HTML5 support for video, Flickr support, iPhone views for the overlay stuff, optimizations, etc.
