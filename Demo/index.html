Flower v1.0

Flower is a JavaScript library designed to give musicians easy access to UI tools like image/video lightboxes and sound players.

All Flower modules have been built to work easily or automatically. The Moviebox lightbox detects links to YouTube, Vimeo, Google Video, MySpace TV, Vevo, and Quicktime video files. The Imagebox module only requires you to add a class to a links or divs â€” same for Soundplayer.

These scripts are built using MooTools and the Soundplayer is powered by the excellent SoundManager 2 by Scott Schiller. For more information and advanced usage please refer to the included README file.

Soundplayer

Soundplayers are created by adding the 'flower_soundplayer' class to a div containing links to audio files. Multiple players are handled intelligently, and by adding 'flower_soundplayer' to a link outside a soundplayer div you can even play links inline, like with this link to "Roka" by Calexico.

There is also a special 'flower_soundplayer_pageplayer' that can be added to a div, gathering all audio links on a page not included in other soundplayers for a play-this-page style player. [open page player] Buke & Gass: Medulla Oblongata (hidden link)
